# alarmbot

A sample [Prague](http://github.com/billba/prague) browser-based bot featuring the classic alarmbot. A good comparison with BotBuilder v3

# How to use

1. clone
2. `npm install`
3. `npm run build` (or `npm run watch` to build on changes)
4. `npm run start`
5. aim your browser at `http://localhost:8000`
